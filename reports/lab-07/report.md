# Лабораторная работа № 7

Меню переведено на загрузку данных с учебного API. При открытии страницы приложение получает список блюд с сервера, сохраняет прежнюю структуру каталога и автоматически показывает локальный резервный набор, если сеть или API недоступны.

Проверки:
- `npx --yes html-validate index.html menu.html order.html` без ошибок
- Playwright-рендер страницы после ожидания загрузки данных с API
